# Meta Tags

Servem para informar determinadas coisas ao navegador.

Ex:

```html
<title>Principal</title>
```

A tag acima altera o nome exibido na aba do navegador.

Exemplo html completo:

```html
<title>Principal</title>

<!--
    A seguir a tag que define o fav icon da aba.
-->

<link rel="icon" href="./image/2991148.png">

<!--
    Meta tag que define a codificação do caractere. A não utilização desta tag pode gerar mal funcionamento nos caracteres especiais em navegadores mais antigos.
-->

<meta charset="UTF-8">

<!--
    Outra meta tag importante define o tamanho da viewport. Pode-se atribuir uma largura fixa, mas isso não será atenderá satisfatoriamente a diversidade de dispositivos, cada um com um tamanho.
    Recomenda-se, portanto, utilizar o seguinte.
-->

<meta name="viewport" content="width=device-width, inicial-scale=1">

<h1>Texto qualquer</h1>
```
