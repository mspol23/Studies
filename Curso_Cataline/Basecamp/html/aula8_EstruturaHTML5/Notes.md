# Estrutura HTML5

```html

<!DOCTYPE html>
<html>
    <head>
        <title>Principal</title>
        <link rel="icon" href="../aula7_Meta_Tags/image/2991148.png">
        <meta charset="UTF-8"/>
        <meta name="viewport" content="width=device-width, initial-stale=1">
    </head>
    <body>
        <h1>Meu Título</h1>
        
        <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Omnis molestias dolor nobis, sunt minus aut perferendis facilis dolore, quam dicta exercitationem repellendus totam? Quo, nostrum? Nesciunt a unde maxime eaque.</p>
    </body>

</html>

<!--
    Para informa ao sistema que esta head e este body pertencem ao mesmo documentos html, coloca-se uma tag com este mesmo nome.

    Por fim temos que informar a versão do html que estamos utilizando com a tag:
    <!DOCTYPE html>

    Atualmente basta informar que se trata de html, que os navegadores já reconhecem que se trata do html5.

    A tag <body> coloca um espaço extra acima do seu conteúdo, mas com CSS resolve-se esta questão.
-->
```
