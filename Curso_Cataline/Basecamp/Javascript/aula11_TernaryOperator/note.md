# Operador Ternário.

Resume o operador if quando houver apenas duas condições.

```js
/ Forma compacta.

var age1 = 18

var result1 = age1 >= 18  ? 'Entre!' : 'Não entre!'
// A interrogação substitui o que seria um "if (true) e os dois pontos estão substituindo o 'else'" 

console.log(result1)
```
